# The Iron Sail

The Iron Sail is a Unity 2D sailing exploration game. You pilot a small boat
through a procedural ocean, follow wind and map clues, manage supplies, trade at
island shops, and survive sea hazards on the way to treasure.

The project focuses on gameplay systems: deterministic world generation,
chunk-streamed islands and hazards, map discovery, save/load, boat handling,
weather, combat, shops, inventory, and responsive UI.
